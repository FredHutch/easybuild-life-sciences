### Fred Hutch R
R with addition libraries from user requests. This release has a new name, placing 'fh' as a prefix. Two modules are created for each R version.
A base R module suited to general purpose use which contains over 700 libraries.
libraries. The Fred Hutch R is a superset of the base R module plus addition libraries. 

### Known Issues
 * Replaces R-4.0.1 which has memory issues. R-4.0.2 is a quick release to fix issues with 4.0.1. Usage of 4.0.1 is discouraged.
